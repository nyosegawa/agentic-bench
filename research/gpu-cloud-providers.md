# GPU クラウドプロバイダ調査 (2026-02)

## 目的

モデルのベンチマーク・推論テストを自動実行するための GPU クラウドを選定する。
重視するポイント:

- 無料枠の大きさ（継続的に無料で使えるか）
- API/SDK による完全自動化のしやすさ
- カスタムスクリプトの実行可否

---

## Tier 1: 無料枠が大きい（常時無料）

### Google Colab (Free)

- **無料枠**: T4 GPU、週 15-30 時間程度、セッション上限 ~12 時間、保証なし
- **有料**: Pro $9.99/月、Pro+ $49.99/月、PAYG $0.10/compute unit
- **API**: 公開 REST API なし。Notebook ベースのみ
- **自動化**: Chrome MCP 経由でブラウザ操作する方針
- **用途**: インタラクティブな実験、プロトタイピング

### Kaggle Notebooks

- **無料枠**: P100 GPU 30 時間/週、20GB ストレージ、セッション最大 9 時間
- **API**: Kaggle API はデータセット/コンペ管理向け。任意 GPU 計算は不可
- **自動化**: 限定的（Notebook ベース）
- **用途**: データサイエンスコンペ、データセットを使った実験

### Hugging Face ZeroGPU

- **無料枠**: H200 スライス（~70GB VRAM）を動的共有。Pro ($9/月) で 7 倍のクォータ
- **有料 Spaces**: T4 $0.40/hr、A10G $1.00/hr、A100 $4.13/hr
- **API**: Inference API あり。Spaces で Gradio/FastAPI エンドポイント公開可能
- **自動化**: 良好
- **用途**: モデルデモ、Gradio アプリ、軽量推論

### SambaNova Cloud

- **無料枠**: Llama 3.3 70B、Llama 3.1 405B、Qwen 2.5 72B 等の推論 API が永続無料（レート制限あり）
- **API**: REST API、OpenAI 互換。自動化に最適
- **制約**: API のみ（任意コード実行不可）
- **用途**: LLM 推論 API の無料利用

## Tier 2: 無料クレジットあり + API 自動化が優秀

### beam.cloud ★現在使用中

- **無料枠**: サインアップ時 10-15 時間分の GPU クレジット（ワンタイム）
- **価格**:
  | GPU | 最安/hr | フル構成/hr |
  |-----|---------|------------|
  | T4 | $0.15 | $0.54 |
  | RTX 4090 | $0.19 | $0.69 |
  | A10G | $0.29 | $1.05 |
  | A100-40GB | $0.63 | $2.25 |
  | A100-80GB | - | $2.81 |
  | H100-80GB | $0.97 | $3.50 |
- **API**: Python SDK + REST API。Webhook、オートスケール、認証組み込み
- **自動化**: 最高。Python 関数をサーバーレスエンドポイントとしてデプロイ
- **備考**: 秒単位課金。現在ずっと無料枠で運用中

### Modal ★要検討

- **無料枠**: $30/月の計算クレジット（毎月リセット）。スタートアップ/学術向けに最大 $25,000
- **価格**: T4 ~$0.59/hr、A10G ~$1.10/hr、A100 ~$3.00/hr、H100 ~$4.89/hr
- **API**: Python SDK、REST API。ファーストクラスのプログラマティックアクセス
- **自動化**: 最高。`@modal.function` デコレータで任意の Python をデプロイ
- **備考**: 秒単位課金。コンテナ、ボリューム、シークレット管理あり。DX が最も優秀

### Lightning AI

- **無料枠**: 月 15 Lightning クレジット（≒ T4 22 時間/月）
- **価格**: T4 $0.68/hr、L4 $0.70/hr、A10G $1.80/hr
- **API**: Lightning AI Studios CLI/SDK
- **自動化**: 良好。ブラウザ内 VS Code 環境
- **用途**: フル開発環境、PyTorch Lightning ワークフロー

### RunPod

- **無料枠**: 常時無料枠なし。初回 $10 課金で $5-$500 のボーナスクレジット
- **価格**:
  | GPU | On-Demand | Spot |
  |-----|-----------|------|
  | RTX 4090 | $0.44/hr | $0.34/hr |
  | A40 | $0.44/hr | $0.29/hr |
  | A100-80GB | $1.64/hr | $1.34/hr |
  | H100-SXM | $3.99/hr | $2.99/hr |
- **API**: REST API、Python SDK、サーバーレスエンドポイント
- **自動化**: 最高。Docker コンテナ、サーバーレスワーカー

## Tier 3: 有料のみ・最安価格帯

### Vast.ai

- **無料枠**: なし
- **価格（マーケットプレイス、変動あり）**:
  | GPU | 最安/hr |
  |-----|---------|
  | RTX 3090 | $0.10-0.20 |
  | A10 | $0.09 |
  | A100-40GB | $0.52 |
  | A100-80GB | $0.80-1.50 |
  | H100 | $1.50-2.50 |
- **API**: REST API、CLI、Python SDK
- **自動化**: 良好。Docker ベース、SSH アクセス

### Lambda Labs

- **無料枠**: なし
- **価格**: H100 PCIe $2.49/hr、H100 SXM $2.99/hr。Egress 無料
- **API**: REST API（インスタンス管理）
- **自動化**: 良好。フル VM（SSH アクセス）

## 自動化との相性ランキング

Claude Code スキルから完全自動化しやすい順:

1. **beam.cloud** - Python SDK 優秀、サーバーレス、既に使用中
2. **Modal** - $30/月無料、SDK 最高品質、`@modal.function` で即デプロイ
3. **RunPod** - REST API 充実、Docker/サーバーレス
4. **Colab** - Chrome MCP 経由（半自動、ブラウザ操作）
5. **Kaggle** - API 限定的、Notebook 縛り
